9. **Harassment & Abuse**
   - Profanity and insults
   - Bullying behavior

## üõ†Ô∏è Admin Actions

### When a Message is Flagged

1. **Review the Message**
   - Read full conversation context
   - Check user history
   - Assess severity

2. **Take Action**
   - **Approve**: If false positive, allow message
   - **Warn**: Send warning to user
   - **Suspend**: Temporary account suspension
   - **Ban**: Permanent account removal
   - **Report**: Report to authorities (if involving minors)

3. **Document**
   - Add notes to user profile
   - Track violation patterns
   - Update moderation rules if needed

### Severity Guidelines

**Level 1 - Warning**
- First-time minor violation
- Borderline inappropriate language
- Unintentional rule breaking

**Level 2 - Temporary Suspension (24-72 hours)**
- Repeated minor violations
- Clear inappropriate content
- Off-platform payment attempts

**Level 3 - Extended Suspension (1-4 weeks)**
- Sexual harassment
- Grooming behavior
- Serious threats

**Level 4 - Permanent Ban + Report**
- Sexual content involving minors
- Repeated grooming attempts
- Severe threats or violence
- Criminal activity
